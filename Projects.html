Technical School: Southern Indiana Career and Technical Center

Semester 1

The main focuses of this semester were computer basics, Python, and logic.

Binary
The year first started off with binary. This got us first introduced (for those that weren't already) with the language in which computers think; my takeaway from this was that it strengthened our logic.
Terminal Navigation
A few weeks after binary we moved onto navigating PowerShell and Vim. It was surprising how well I took to Vim; coming from a file-explorer-only perspective it was fascinating how much control I had over my local machine and quickly I was able to edit and create new files with just a few keystrokes.
Python
We are still using Python to this day due to its versatility and capabilities. In my opinion it is the best gateway language into programming as its simple logic and syntax are perfect for beginners.
Iterating through lists, loops, functions, and conditionals were a huge part of what we did. I have experience in python prior to taking this course but this made me realize I knew very little before SICTC; every day brings about that realization more and more.
Turtle Python
This was one of the most anguishing parts of the first semester. It wasn't hard by any means, it was simply long and exhausting. Nothing drove home the concept of DRY (Don't Repeat Yourself) code more than this. Loops are essential to TurtleGUI programming.
Finals
Fast Food Ordering System
Utilizing loops and list iterations, my class was tasked with creating an ordering system for the Krusty Krabs from SpongeBob. I had fun building this.
Build Your Own Application
The app built by Mason Durcholz and I, using block-based coding, was one that allowed the user to store reminders. It could hold up to five (we intended to add more but ran out of time) and featured a dark mode for user-accessibility.

Semester 2

The main focuses of this semester were HTML, CSS, and MySQL databases.

HTML
I learned HTML (HyperText Markup Language) through my teacher's instruction and the FreeCodeCamp Responsive Web Development Course.
CSS
We used CSS (Cascading Stylesheets) in the making of several class websites, including this one. CSS is extremely useful and practically essential for the customization and user experience of a website.
MySQL
MySQL (Structured Query Language) expanded my understanding and what I thought was possible for file manipulation. Not only can I pull information remotely from a database, but it also enabled me to directly edit the files too using JSONs and QueryStrings.
Finals
Personal Website
It's this website! This is where I can use all of my knowledge on HTML and CSS to the fullest and create a portfolio I can truly be proud of.
Build Your Own Application Revision
Using the block-based applications we made for one of the finals in the first semester and Google forms we used to take turns reviewing each other's apps, we are tasked with going back and choosing someone else's app to revise, fix, and improve.
Galactic Registry
Our task was to build a MySQL database using MariaDB. It featured many complex techniques such as various joins to display specific tables, using a staging table to import select data into other tables.

Challenges

I would say that my biggest struggles this semester definitely came from data analytics and MySQL.

The unfamiliarity of the two threw me for a loop and proved to be very frustrating. Data analytics for the methods for interacting with the CSV; pulling data, using Python to manipulate it, etc.

MySQL was extremely confusing for me, I understand it for the most part, though it is difficult to make sense of joins

What I've Learned

I am glad I decided to enroll in this class. It has not only bestowed upon me a wealth of knowledge but widened my understanding and ability to learn.

If I were to liken anything to it, it would be throwing yourself at a brick wall over and over again until you eventually break through. Every attempt helps you break a little more each time.
