Technical School: Southern Indiana Career and Technical Center

Semester 1

Binary
The year first started off with binary. This got us first introduced (for those that weren't already) with the language in which computers think; my takeaway from this was that it strengthened our logic.
Terminal Navigation
A few weeks after binary we moved onto navigating PowerShell and Vim. It was surprising how well I took to Vim; coming from a file-explorer-only perspective it was fascinating how much control I had over my local machine and quickly I was able to edit and create new files with just a few keystrokes.
Python
We are still using Python to this day due to its versatility and capabilities. In my opinion it is the best gateway language into programming as its simple logic and syntax are perfect for beginners.
Iterating through lists, loops, functions, and conditionals were a huge part of what we did. I have experience in python prior to taking this course but this made me realize I knew very little before SICTC; every day brings about that realization more and more.
Turtle Python
This was one of the most anguishing parts of the first semester. It wasn't hard by any means, it was simply long and exhausting. Nothing drove home the concept of DRY (Don't Repeat Yourself) code more than this. Loops are essential to TurtleGUI programming.
Finals
Fast Food Ordering System
Utilizing loops and list iterations, my class was tasked with creating an ordering system for the Krusty Krabs from SpongeBob. I had fun building this.
Build Your Own Application
The app built by Mason Durcholz and I, using block-based coding, was one that allowed the user to store reminders. It could hold up to five (we intended to add more but ran out of time) and featured a dark mode for user-accessibility.
